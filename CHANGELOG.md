## [v0.1.0] - 2026-04-19

### Highlights

- Spec-Driven Development Evaluation CLI using LLM-as-a-Judge
- Built-in support for Kiro (AWS), SpecKit (GitHub), and PlexusOne SDD methodologies
- Multi-provider LLM support via OmniLLM: Anthropic, OpenAI, AWS Bedrock, Google Gemini
